# Suggested commands (Windows)
- Install deps: `pnpm install`
- Build: `pnpm build`
- Unit tests (stable): `pnpm exec jest --config jest.config.cjs --runInBand --forceExit`
- Single test file: `pnpm exec jest --config jest.config.cjs tests/unit/ToolDefinitions.test.ts --runInBand`
- Verify tool definition vs handlers: `pnpm verify`
- Integration smoke: `node test_integration.mjs`
- Live URL smoke: `node scripts/live-smoke.mjs`
- Dev run: `pnpm dev`
- Start built server: `pnpm start`
- Lint: `pnpm lint`
- Format: `pnpm format`
